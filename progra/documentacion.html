Introducción

Este proyecto representa uno de mis primeros acercamientos formales al desarrollo web. A través de la creación de esta página, he aprendido las bases del lenguaje HTML, entendiendo su estructura, etiquetas principales y su utilidad para crear sitios web. Esta experiencia no solo me ha permitido desarrollar habilidades técnicas, sino también comprender cómo organizar información de forma clara y aun que html es muy limitado para diseñar podesmos aprender lo primordial con ciertas etiqutas y esturcturas.

Objetivos

Objetivo general:

Aprender y aplicar correctamente las principales etiquetas y estructuras del lenguaje HTML en un proyecto web básico.

Objetivos específicos:
  • Comprender la función de etiquetas fundamentales como html>, head>, body>, meta>, title>, h1> a h6>, p>, entre otras.
  • Utilizar correctamente la etiqueta meta charset="UTF-8"> para mostrar correctamente caracteres especiales como la ñ o tildes.
  • Diseñar una estructura básica de una página web con secciones como carátula, introducción, objetivos, contenido, conclusión, observaciones y bibliografía.

Contenido

Dentro de esta estructura, usé la etiqueta head> para incluir elementos importantes como el título del sitio y los metadatos.

meta charset="UTF-8"> También agregué la etiqueta title> para definir el título que aparece en la pestaña del navegador: html title>Mi página personal en HTML/title> En la parte del body>, estructuré las secciones de mi proyecto usando etiquetas como h1> para los títulos principales, h2> y h3> para subtítulos, y p> para los párrafos. Conclusión

Este proyecto ha sido una excelente oportunidad para conocer el lenguaje HTML ya que tenia cero conocciemtos de html. Aprendí cómo estructurar una página correctamente, cómo usar etiquetas básicas y cómo organizar el contenido en secciones claras. A pesar de que es un lenguaje sencillo en su forma básica, HTML es muy poderoso para presentar información de manera ordenada, me siento más seguro en mi proceso de aprendizaje y motivado a seguir avanzando hacia CSS que me llama mucho la atencion ya que se pude diseñar hermosas paginas por lo que tengo entendido y otros lenguajes más avanzados.

Observaciones

Durante la realización de este proyecto, comprendí que HTML requiere atención a los detalles, ya que un pequeño error en la estructura puede afectar la visualización de la pagina.
